Firebase Setup (Secure):

  1. Go to Firebase Console
  2. Create project ‚Üí Realtime Database ‚Üí Create Database
  3. Copy database URL (e.g., https://xxx.firebaseio.com)
  4. Go to Project Settings ‚Üí General ‚Üí Web API Key
  5. Go to Authentication ‚Üí Sign-in method ‚Üí Enable "Anonymous"
  6. Set database rules to: 
    {
  "rules": {
    "users": {
      "$uid": {
        ".read": "$uid === auth.uid",
        ".write": "$uid === auth.uid"
      }
    },
    "sync": {
      "$syncId": {
        ".read": "auth != null",
        ".write": "auth != null"
      }
    }
  }
}

Cross-Device Sync: Set the same Sync ID on all your devices/profiles to share data. The sync path allows any authenticated user with the same Sync ID to read/write.
